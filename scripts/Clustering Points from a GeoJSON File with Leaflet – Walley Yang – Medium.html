Clustering Points from a GeoJSON File with Leaflet

Recently, I’ve been tinkering with Leaflet and GeoJSON. I started to experience slowness and lag while trying to display several thousand points from datasets, which makes sense because my computer will have to process all the information. For this example, I went to MapZen and extracted 5581 points from a small area in Washington D.C. You will probably experience slowness while processing all those points as you zoom in and pan around the map.

Luckily, there is the Leaflet.markercluster plugin to help with clustering all those points! What I did was created a plugin folder for the Leaflet.markercluster plugin and added the stylesheet and JavaScript files to my HTML page.

I then added the markercluster layer and used jQuery to get the data points from the sample GeoJSON file. All that’s left afterwards is to add the layer with the data points to the markercluster layer and add it to the map.

You can now see that all the previous points are now in clusters and will separate from the clusters as you zoom in. Implementing the Leaflet.markercluster plugin greatly improved the speed of processing the information. Feel free to check out the live example and repository.
